Interface ILanguageSettings

  • public interface ILanguageSettings
    Allows the client to configure which languages will be included in downloads for HLS and DASH asset types The currently configured LanguageSettings can be accessed through the ISettings.getLanguageSettings call in ISettings. Modified settings are only applied once you call ISettings.setLanguageSettings
