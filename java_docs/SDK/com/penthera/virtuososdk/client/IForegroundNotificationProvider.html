Interface IForegroundNotificationProvider

  • public interface IForegroundNotificationProvider
    This interface defines the methods which must be provided if the client application wishes to generate update notifications from within the service process. A class implementing this interface must be returned by VirtuosoForegroundServiceHandler.getForegroundServiceNotificationProvider(). The implementing class must have a default constructor and will be instantiated by the SDK within the VirtuosoService. It will then be used from within the service process to request updates to the app foreground notifications.
