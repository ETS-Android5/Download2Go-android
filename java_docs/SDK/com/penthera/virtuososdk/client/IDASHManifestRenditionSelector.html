Interface IDASHManifestRenditionSelector

  • public interface IDASHManifestRenditionSelector
    Virtuoso will automatically select video renditions based on the supplied filters and maximum bitrate provided during asset creation. If you have more advanced requirements for rendition selection of an MPEG DASH asset that cannot be fulfilled by the automatic selection process, you may define your own selection by implementing a IDASHManifestRenditionSelector and registering it with the background processing manager. Prior to selecting the rendition for download, the SDK will provide the IDASHManifestRenditionSelector a list of IDASHVideoRendition objects. You may choose which one you would like to use for download, and return that to the SDK.

      Given a list of all the available video representations within a DASH AdaptationSet, return the required representation for download. The method should be implemented to return one of the values from within the list. If another object, or null is returned then the video AdaptationSet will be removed from the asset.
