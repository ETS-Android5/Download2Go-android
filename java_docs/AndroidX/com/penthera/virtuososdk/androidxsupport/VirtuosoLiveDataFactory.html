    The Penthera AndroidX support library provides lifecycle aware LiveData components that are designed to be used within the ViewModel of a modern android app, following the Application Architecture Components design.
    The library consists of a series of independent LiveData wrappers for common SDK objects, each of which can be created through the VirtuosoLiveDataFactory:
    • SegmentedAssetLiveData provides a LiveData wrapper for a single asset, which will update inline with asset state changes and download progress which come from the downloader service
    • AssetListLiveData provides a LiveData representation of every asset currently managed by the SDK.

    In order to create lifecycle aware components, a lifecycle owner must be provided in construction of the factory. This will be used throughout construction of the requested data. The factory contains a lifecycle observer, linked to the Virtuoso class which underpins the work of the factory. It is recommended to use a single Virtuoso instance within your application or activity. Either provide an instance to the factory upon construction, or construct the factory first and get the Virtuoso object from the factory for your other needs.
    The lifecycle owner can be either an Activity/Fragment or a ProcessLifecycleOwner.
