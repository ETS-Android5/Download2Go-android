  • public class EnginePauseHelper extends androidx.lifecycle.LiveData<EnginePauseHelper.EnginePauseState>
    A helper class describing the current engine paused state and providing methods to pause and resume downloads on the engine.
    The reported value of this helper will be one of 5 states, from the EnginePauseState enumeration:
    • UNSET - this will only occur during initialization, prior to a value becoming available from the service.
    • RESUMED - the engine is not paused and will download when required.
    • PAUSING - a pause has been requested via the pauseEngine method but the engine has not yet completed the action.
    • PAUSED - the engine is paused and will not download any content.
    • RESUMING - a resume has been requested via the resumeEngine method but the engine has not yet completed the action.
