Class ExoplayerUtils

    • com.penthera.virtuososdk.support.exoplayer212.ExoplayerUtils
  • public class ExoplayerUtils extends java.lang.Object
    Utility class that provides functionality to more easily link the Virtuoso SDK to Exoplayer version 2.11.* This class provides the ability to fully initialize a SimpleExoPlayer instance for playback or to create the various components needed, (DrmSessionManager, MediaSource), to easily integrate playback of offline content through the Virtuoso SDK.

      Create and initialize a SimpleExoPlayer instance for playback of the specified asset using the provided options This method will create a SimpleExoPlayer instance, link it to the provided PlayerView and initialize it for playback with a MediaSource that points to the passed in IAsset.       options - ExoplayerUtils.PlayerConfigOptions used to configure the player, media source and drm for playback.
